# Week 2 – Exploratory Data Analysis(EDA)

# Project overview
The objective of this project is to perform Exploratory Data Analysis (EDA) on the given dataset to identify patterns, trends, distributions, and extract meaningful insights.


# Objectives
-> Understand dataset structure
-> Perform descriptive statistics
-> Identify missing values and duplicates
-> Detect trends and outliers
-> Visualize data for better understanding


# Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

* TotalPrice
